The problems tab will show any known syntax errors in your <DbtCoreR /> Project. The errors are grouped by files. Clicking on the error will jump directly to the error.

<img src="/product/problems.png"></img>


## Ignore types of problems
You can ignore all instances of a type of problem by right clicking on an instance of the problem and clicking **Ignore problems like these**.
<img src="/product/ignore-problem-types.png"></img>

## Reset ignored problems
You can reset an ignored problem by right clicking by on an instance of the problem and clicking **Stop ignoring problems like these**.
<img src="/product/reset-ignored-problems.png"></img>
