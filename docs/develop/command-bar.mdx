Click the input bar at the bottom of the app or press <Keybinding mac="⇧ ⌘ D" winux="ctrl+shift+d" /> to run a <DbtCoreR /> command.

The Command Bar offers rich autocomplete suggestions for <DbtCoreR /> commands and arguments, model names, project directory paths, and file names.

<img src="/product/command-bar.png"></img>

## Command History
See a history of previous commands and their output in the Command History window.

<img src="/product/command-history.png"></img>

### Rerun Command
Easily rerun a previous command by hovering over the command clicking **Rerun command**.
<img src="/product/rerun-command.png"></img>
